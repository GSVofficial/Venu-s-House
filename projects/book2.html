CSS INTRODUCTION

  • CSS stands for Cascading Style Sheet.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.

Example of CSS

CSS SELECTORS

  • CSS selectors are used to "find" (or select) the HTML elements you want to style.
  • Simple selectors (select elements based on name, id, class).
  • Combinator selectors (select elements based on a specific relationship between them).
  • Pseudo-class selectors (select elements based on a certain state).
  • Pseudo-elements selectors (select and style a part of an element).
  • Attribute selectors (select elements based on an attribute or attribute value).

CSS BORDERS

  • Borders are used to create lines around HTML elements, giving them visual boundaries.
  • Borders can be styled using width, style, and color properties.
  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border
  • groove - Creates a 3D grooved border
  • ridge - Creates a 3D ridged border
  • inset - Creates a 3D inset border
  • outset - Creates a 3D outset border
  • none - No border
  • hidden - Hides the border

CSS MARGINS

  • The CSS margin properties are used to create space around elements, outside of any defined borders.
  • With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

    • Margin Declaration Types

  • margin-top
  • margin-bottom
  • margin-left
  • margin-right

CSS PADDING

  • The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
  • With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element (top, right, bottom, and left).

    • Padding Declaration Types,

  • padding-top
  • padding-bottom
  • padding-left
  • padding-right

CSS DISPLAY PROPERTIES

  • The display property is used to specify how an element is shown on a web page.
  • Every HTML element has a default display value, depending on what type of element it is. The default display value for most elements is block or inline.
  • The display property is used to change the default display behavior of HTML elements.

    • There are types of display properties

  • block
  • inline
  • inline-block
  • flex
  • grid

    • CSS ANIMATIONS

    • CSS allows animation of HTML elements without using JavaScript!
    • CSS animations allow you to create smooth transitions between different styles on HTML elements without using JavaScript. With CSS, you can animate things like position, color, size, opacity, and more
    • You can change as many CSS properties you want, as many times as you want.
    • To use CSS animation, you must first specify some keyframes for the animation.
    • Keyframes hold what styles the element will have at certain times.
