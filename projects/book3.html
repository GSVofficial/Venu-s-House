    JAVASCRIPT DECLARATIONS

      How to declare?

    • In JavaScript, you can declare variables in the below three ways:
    • 1. var variableName - old way, function scoped.
    • 2. let variableName - modern way, block scoped.
    • 3. const variableName - constant, block scoped.

    By var

  • They can be re-declared.
  • They do not have a fixed value.

    • By const

  • They are fixed values.
  • Cannot be updated or re-declared.

    • By let

  • Can be updated, but not re-declared in the same scope.
  • Has block scope (like inside { ... })
  • Preferred when the value will change.
